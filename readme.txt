The sample project consists of the following components:

1. BackgroundService that extends Android AccessibilityService

2. AndroidManifest.xml (generated by Android Studio) that specifies permissions and 
   brings up the Android Accessibility Settings window to enable the Alien Keyboard Wedge service

3. phone_accessibility.xml that specifies the type of key events to capture

4. HomeActivity is a dummy activity

Installation steps:

1. Create and install the keyboard service .apk

2. In order to enable the service the Settings -> Accessibility -> Alien Key Wedge must be turned on
   Starting the app will automatically bring up the Accessibility Setting window

3. Pressing the SCAN button will read and display the tag information
